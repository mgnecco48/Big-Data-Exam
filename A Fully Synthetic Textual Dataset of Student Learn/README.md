# A Fully Synthetic Textual Dataset of Student Learning Habits and Preferences Generated Using a Large Language Model

## Description

This repository contains a fully synthetic textual dataset generated using the ChatGPT web interface (GPT-4.1, 2025 release). Model parameters such as temperature and top-p were not user-configurable in the web interface and therefore remained at system-default settings. The dataset simulates fictional student learning habits, preferences, challenges, and opinions related to online education. No real individuals, surveys, or proprietary data sources were used in the creation of this dataset.

The dataset is intended for **research benchmarking**, **natural language processing (NLP)**, **educational data mining**, **survey analysis**, and **machine learning experimentation**.

---

## Synthetic Nature of the Dataset

All records in this dataset are artificially generated. The dataset does not represent real individuals or real-world survey results. The use of synthetic data eliminates ethical, privacy, and copyright concerns while supporting reproducibility and open research.

---

## Dataset Contents

* **File name:** `synthetic_student_learning_dataset_10000.csv`
* **Number of records:** 10,000
* **File format:** CSV
* **Language:** English

---

## Column Description

| Column Name                 | Description                             |
| --------------------------- | --------------------------------------- |
| `respondent_id`             | Unique synthetic identifier             |
| `education_level`           | Academic level of the fictional student |
| `study_hours_per_day`       | Daily study hours (1–6)                 |
| `preferred_learning_method` | Preferred learning method               |
| `main_learning_challenge`   | Primary learning difficulty             |
| `motivation_level`          | Learning motivation level               |
| `online_learning_opinion`   | Opinion about online learning           |
| `device_used_for_study`     | Primary device used for learning        |

---

## Data Generation Method

The dataset was generated using ChatGPT web interface (GPT-4.1, 2025 release) following a structured prompting strategy. Controlled vocabularies and predefined constraints were applied to ensure consistency, diversity, and realism across records.

No external datasets, human responses, or copyrighted materials were used during generation.
